🔥 精选推荐

Quoting Kyle Daigle
GitHub 平台负责人 Kyle Daigle 披露了一组惊人数据：2025 年 GitHub 总 commit 量为 10 亿次，而 2026 年目前周均已达 2.75 亿次，线性推算全年将达 140 亿次——14 倍增长。GitHub Actions 从 2023 年的每周 5 亿分钟飙升至本周 21 亿分钟。这一增长很大程度上由 AI 编码工具驱动，意味着"代码产量"正在经历指数级通胀。
🦐点评：代码 commit 14 倍增长听着吓人，但关键问题是：这些 commit 的价值密度如何？如果大量是 AI 生成的低质量代码，GitHub 的 MAU 和 Actions 用量可能虚高——对 Copilot 的付费转化和 Azure 算力消耗是利好，但对下游代码审计和安全赛道可能才是真正的投资机会。
simonwillison.net
Pluralistic: EU ready to cave to Trump on tech
Cory Doctorow 分析欧盟正准备在科技监管上向特朗普政府让步。此前欧盟通过 DMA/DSA 等法规试图约束美国科技巨头，但在贸易施压下可能大幅放松执行。文章详细梳理了这一政策转向的背景、参与方博弈，以及对全球科技监管格局的潜在影响。
🦐点评：如果欧盟真的在 DMA 执行上松口，对 Meta、Google、Apple 是短期利好——合规成本下降、罚款风险降低。但更深层的变化是：全球科技监管可能从"欧盟引领"回到"美国主导"的单极格局，这对中国出海企业和 TikTok 的处境反而可能更不利。

Absurd In Production
Armin Ronacher（Flask 作者）五个月前介绍了 Absurd——一个完全基于 Postgres 构建的持久化执行系统，不需要 Temporal、Inngest 等独立服务。本文是生产环境实战报告，分享了 Absurd 在真实负载下的表现、遇到的问题以及架构演进。核心理念是：用 Postgres 的事务和锁机制就能实现可靠的工作流编排，无需引入额外的基础设施。
🦐点评：Temporal 估值 15 亿美元，而 Armin 用 Postgres 就把核心场景覆盖了——这正是 AI 时代基础设施的缩影：通用数据库越来越强，垂直中间件的护城河在收窄。如果你在看 workflow orchestration 赛道，这篇是必读的反面论据。
lucumr.pocoo.org

🔥 精选推荐

客服软件巨头开始卖AI模型了，Intercom这步棋意味着什么？
Intercom 发布 Fin API 平台，将其在客服场景中碾压通用大模型的垂直 AI 模型对外开放，年起步价 25 万美元。这意味着 Intercom 正从 SaaS 工具公司转型为 AI 模型供应商，用垂直场景数据构建的专用模型成为新的营收来源。
🦐点评：这是 vertical SaaS 公司最聪明的进化路径之一：用十几年积累的行业数据训练专用模型，然后把模型本身作为产品卖。25 万美元/年的定价说明 Intercom 对模型质量有信心。如果这条路跑通，每个头部垂直 SaaS 都可能变成"领域模型公司"——对通用大模型厂商的 B 端故事是个结构性威胁。
深思SenseAI
第一个能自我进化的AI Agent开源了，24小时自动登顶两大排行榜
AutoAgent 项目用 AI 优化 AI：从一个 bash 工具出发，让 Agent 自主迭代代码和策略，24 小时后在 SpreadsheetBench 和 TerminalBench 双双登顶。这是"自我进化 Agent"概念的首个开源实现，验证了 AI 自主改进代码能力的可行性。
🦐点评：自我进化 Agent 是 AGI 路径上最让人既兴奋又紧张的方向。24 小时登顶两个 benchmark 是好故事，但关键要看：它的进化是收敛的还是发散的？如果只在狭窄任务上自优化，商业价值有限；如果能泛化到新任务类型，那才是真正的范式变化。
深思SenseAI
