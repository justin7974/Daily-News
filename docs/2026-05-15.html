[AINews] Codex Rises, Claude Meters Programmatic Usage
GPT 5.5 发布三周以来，AI 工程师群体中 Codex 的风评明显上扬——模型实力接近 Mythos 级别、Codex for Everything Else 上线、加上更慷慨的 API 额度，三重利好叠加。同日 Anthropic 宣布 Claude 订阅将"交互用量"与"程序化用量"分离：6 月 15 日起，每个付费订阅按月费金额发放等额 API credits（$200 套餐 = $200 credits），用于 claude -p、Agent SDK、第三方 harness 等自动化调用场景。此前这些用量享受估算 70-90% 的隐性补贴，新政实质上取消了这一补贴。OpenAI 同日推出企业用户"30 天内切换到 Codex 即送 2 个月免费"的促销。
🦐点评：Claude 的定价调整和 OpenAI 的企业抢客促销在同一天落地，编码 Agent 赛道正式进入"抢地盘"阶段。对 OpenClaw、Cline 等第三方 harness 而言，这是一个结构性转折——免费搭 Claude 订阅便车的日子结束了，要么引导用户转投 Codex/GPT 5.5，要么开始自建商业模式。对 VC 来说，这验证了一个判断：AI 实验室从来不打算让 harness 层成为独立利润中心。

From "System of Record" to "System of Intelligence"
a16z 团队在昨日 Seema Amble 的"Is Software Losing Its Head?"之后发出续篇。核心类比：CRM 正在经历类似社交网络的权力迁移——Facebook 的"好友图谱"曾是最有价值的资产，但 News Feed 算法出现后，图谱降级为"众多输入之一"。CRM 也正在从"你打开的系统"变成"Agent 在后台调用的 API"，真正的价值层上移到推理层（System of Intelligence）。文章认为新一代公司的壁垒在于：拥有行动执行能力（不止存储，而是能代用户完成操作）、专有数据飞轮、以及多方网络效应。
🦐点评：连续两天 a16z 密集输出 SoR 主题，信号很明确——他们正在系统性地重估企业软件赛道。社交网络类比特别有力：MySpace 的个人主页被 Facebook Feed 颠覆时，没人觉得"个人主页"会消亡，它只是变得不重要了。同样，Salesforce 不会消亡，但它的战略地位正在从"不可替代的核心"降格为"可插拔的存储层"。VC 的 action item：重新审视 portfolio 中 SaaS 公司的护城河——如果壁垒主要来自 UI 粘性和肌肉记忆，Agent 时代可能会加速其侵蚀。
a16z.news
First Public macOS Kernel Memory Corruption Exploit on Apple M5
安全研究团队 Calif 宣布：借助 Mythos Preview，他们在 5 天内构建出首个公开的 M5 芯片 macOS 内核内存损坏漏洞利用。Apple 花了 5 年时间打造 MIE（Memory Integrity Enforcement）——一套硬件级内存完整性保护方案，被多数安全专家视为消费级设备最强防线。研究团队已在 Apple Park 现场向 Apple 提交了漏洞报告。
🦐点评：5 年防御 vs 5 天攻破——这不是 Apple 做得不好，而是 AI 正在从根本上改变攻防经济学。Mythos Preview 的安全研究能力已被英国 AISI 和这个独立团队同时验证。对投资人来说有两层含义：(1) AI-powered 安全攻防工具将成为下一个爆发品类，传统渗透测试公司面临"10x 效率提升或被淘汰"的抉择；(2) 硬件安全投资逻辑需要更新——当 AI 能在天级别破解年级别的防御工程时，"硬件不可变"反而成了劣势。
calif.io
AI-Native Healthcare: 100M Doctor Visits, 10–20 Hours Saved
Abridge 成立于 2018 年，远早于 ChatGPT 时代，从医患对话记录切入，把临床文档自动化做成了医疗 AI 的标杆。公司预计今年覆盖 8000 万以上医患对话，服务 250 家大型美国医疗系统，支持 28 种语言和 50 多个专科。2025 年 6 月以 53 亿美元估值完成 3 亿美元融资。创始人 Janie Lee 是医生出身，深度理解 EHR 工作流、保险预授权和合规要求，这些领域知识构成了极高的产品壁垒。
🦐点评：Abridge 是"AI-native vertical"最好的注脚——不是等 GPT 出来再做 wrapper，而是先花 4 年打磨 healthcare workflow 理解力，然后用 LLM 做 10x 加速器。53 亿估值对应 250 家头部医院客户，单客 ARR 推算在数十万美元量级。更重要的是，医患对话是最难获取的数据类型之一，Abridge 的数据飞轮一旦转起来，后来者几乎无法追赶。这也解释了为什么垂直 AI 的估值天花板在 2025-2026 年持续上移。

Meta's New Reality: Record High Profits. Record Low Morale
WIRED 采访十多位 Meta 现任和前任员工，揭示公司内部士气处于历史最低点。下周三（5 月 20 日）Meta 将裁掉约 10% 员工。一位 Instagram 团队成员称："每个人都不开心，唯一不不开心的人是高管。"这一切发生在 Meta 创下利润历史新高的背景下——公司正在把 AI 投入当作绝对优先事项，人员优化成为系统性策略。
🦐点评：利润历史新高 + 士气历史最低——这是 Zuckerberg 主动选择的"效率偏执症"模式，和 2023 年的"效率之年"如出一辙。对 VC 来说关注两个信号：(1) Meta 10% 裁员意味着又一批高级 AI/ML 人才流入市场，创业和人才招聘窗口再次打开；(2) Meta 把利润全部回灌 AI，表明大厂已进入"不计短期成本争夺 AI 主导权"阶段——这对依赖大厂 API 定价的中小 AI 公司既是利好（成本可能继续降）也是威胁（大厂随时可以做你的功能）。

打破AI融资纪录？Anthropic估值超9000亿美元，拟融资300亿
据知情人士透露，Anthropic 正就筹集至少 300 亿美元的新一轮融资进行早期谈判，估值超过 9000 亿美元（不含本次投资）。此轮融资预计最早本月底完成。同时公司最快 10 月 IPO。Anthropic 同日发布 12 款法律 AI 工具，通过 Claude Cowork 向付费用户提供商业合同审查、律师考试备考等功能，并与 DocuSign、Thomson Reuters 及竞品 Harvey 打通集成。
🦐点评：9000 亿美元估值意味着 Anthropic 已经是全球最贵的未上市公司之一，和 SpaceX 在同一梯队。300 亿融资若在月底落地，将一举超过 OpenAI 此前的所有融资纪录。但关键数字是 10 月 IPO——这个时间表比市场预期早得多，说明 Anthropic 的现金消耗速度和算力投资需求已经到了必须走公开市场的阶段。法律垂直工具的推出则是估值故事的关键注脚：证明 Claude 不只是开发者工具，而是能切入高单价专业服务市场的平台。

Salesforce 主动砍掉了界面，a16z 合伙人说：护城河从来不在那里
深度解读 Salesforce 推出 headless 产品和 a16z 合伙人 Seema Amble 文章的投资含义。核心问题：当软件去掉了界面，剩下的东西和一个 PostgreSQL 加一层 API 有什么本质区别？Seema Amble 认为传统 SoR 的护城河不在 UI，而在运营逻辑、合规数据和多方网络效应——但这些在 Agent 时代是否依然成立，是一个开放问题。
🦐点评：这篇和今天 RSS #2 的 a16z 文章互为补充——RSS 那篇讲"CRM 变成输入层"，这篇追问"输入层还值钱吗"。对 VC 最直接的启示：SaaS 估值体系中"高切换成本"这个变量正在被重新定价。如果 Agent 可以无缝在不同 SoR 之间切换，那么客户锁定率这个指标的含金量就大幅缩水。下一波 SaaS 并购中，买方出价逻辑将从"用户数 × ARPU"转向"专有数据量 × 数据独占性"。
深思SenseAI
Gamma 联合创始人：你以为只有应届生在怕 AI？大模型公司也一样
Gamma App 联合创始人 Grant Lee 打破了"AI 恐慌从下往上递减"的预设——他的观察是：不确定性贯穿基础模型公司、应用层创业公司、传统巨头、知识工作者、应届生每一层，每层的恐慌同时存在，只是原因不同。基础模型公司担心下一代模型淘汰自己的投入，应用层怕被上游吞并，巨头怕组织惯性跟不上，没有人站在高地上俯视。
🦐点评：这篇的独特价值在于提供了一个"全层级恐慌地图"——当一个 AI 应用公司创始人公开承认自己也在怕时，说明市场还远没到"格局已定"的阶段。对 VC 来说这是个重要校准：不要因为某个赛道看起来"已有赢家"就不投——在每一层都不确定的环境里，格局翻转的概率比稳态市场高得多。Grant Lee 的坦诚也侧面说明 Gamma 的估值逻辑：不是"AI PPT 工具"这么简单，而是在押注一个所有人都还在找方向的市场。
