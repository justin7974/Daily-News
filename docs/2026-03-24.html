There Are Only Two Paths Left for Software
a16z 合伙人 David George 发出公开信：软件行业的"舒适中间地带"已经结束。公开市场已完成重新定价，SaaS 公司只剩两条路——要么在 12-18 个月内通过 AI 原生产品将营收增速提升 10 个百分点以上，要么重组公司达到 40%+ 真实经营利润率（含 SBC）。他建议 CEO 找出组织里最强的 5 个人（不论职级），用他们领导信息采集冲刺，然后据此淘汰一半高管团队，将 50% 研发资源投入全新 AI 产品，采用四人小组制从第一天开始写代码。
🦐点评：这篇文章的核心不是"AI 很重要"——而是 a16z 在暗示自己投资组合里的 SaaS 公司必须做二选一。Path 2 的"重组到 40% 利润率"意味着大规模裁员和 AI 替代人力，这对 HR Tech、企业培训等赛道的需求侧是毁灭性的。反过来，能帮 SaaS 公司走 Path 1 的 AI 基础设施（数据层、eval 工具、AI-native 开发平台）可能迎来一波急切的企业客户。

Nvidia's New Server Rack Will Run AI Chips Made by Rivals
Nvidia 在 GTC 大会上宣布其新一代服务器机架将支持竞争对手的 AI 芯片。尽管面临 AMD、Intel 和各云厂商自研芯片的竞争加剧，Nvidia 找到了一种即便从对手芯片上也能赚钱的方式——通过将自身定位为 AI 基础设施平台而非单纯的芯片供应商。
🦐点评：这是 Nvidia 从"卖铲子"到"卖矿场"的战略升级。当你的服务器机架成为行业标准、连竞品芯片都跑在你的 rack 上时，护城河就从硅片转移到了生态。Jensen 在用 Android 的逻辑做硬件——开放平台锁定开发者，比封闭系统更难被替代。对 AMD/Intel 来说，表面上多了一个出货渠道，实质上更深地嵌入了 Nvidia 的生态依赖。

至简贾鹏：最快具身独角兽半年融资 20 亿，做"六边形战士"才能活
至简动力创始人贾鹏首次深度受访。他此前是理想汽车智驾技术研发负责人，2025 年底离职创业，半年内完成 5 轮融资共计 20 亿元，投资方包括元璟、蓝驰、红杉中国、腾讯和阿里。贾鹏判断具身智能会复刻自动驾驶的演进路径：最终活下来的一定是软硬件深度集成的"六边形战士"，而非纯算法供应商。他坚持大一统模型、真实数据优先（合成数据有用但不是主力），先在小场景闭环再扩张。李想对他说："行业里第一个做的通常会死，但最后成功的一定是第一批。"
🦐点评：这篇访谈最有价值的信息不是至简本身，而是贾鹏排除其他方向的逻辑——大模型是大公司游戏、AI Agent 门槛低易红海、AI 硬件偏 C 端运营。这个排除法如果成立，说明具身智能可能是当前中国 AI 创业中少数同时具备技术壁垒和资本密度的赛道。但 20 亿融资之后还没有明确的商业化场景，"先建 Infra 再找场景"的策略风险不低——自动驾驶走这条路花了十年。

企业养虾🦞（OpenClaw），虾住在哪里比虾本身重要
文章从企业 Agent 落地角度提出核心论点：模型能力不是瓶颈，数据环境才是。企业需要构建四层知识体系——数据、数据间的关系、操作权限、Agent 技能。Skill 的质量上限取决于底层三层的完整度，底层是空的，Skill 再精致也是空转。作者以飞书为案例，分析其在"给 Agent 提供工作环境"方面的优势：结构化数据+权限系统+API 生态。核心观点：模型是租来的，知识才是自己的。
🦐点评：这篇文章对企业 Agent 的分析框架很实用——"虾住在哪里比虾本身重要"这个比喻精准地指出了当前企业 AI 落地的真正卡点。对 VC 来说，这意味着投 Agent 公司不如投帮企业建"Agent 栖息地"的公司——数据中间件、权限管理、知识图谱工具。飞书作为案例也暗示，在中国市场，协同办公平台可能比独立 Agent 创业公司更容易吃到企业 AI 的红利。

速递｜OpenAI 广告首测暴露不足，预算消耗仅达承诺额的 15%–20%
OpenAI 在 ChatGPT 中进行的首批广告测试效果不及预期，广告主的预算消耗仅达到承诺额的 15%-20%。这暴露了对话式 AI 产品在广告变现上的结构性挑战——用户意图明确、交互短平快，与传统信息流广告的"漫游式浏览"场景天然矛盾。
🦐点评：15%-20% 的预算消耗率对首测来说是一个很差的信号。Google 搜索广告之所以印钞，是因为用户在"寻找答案"的过程中会看到广告；但 ChatGPT 直接给答案，用户根本没有"浏览"环节。这可能意味着 OpenAI 的广告收入天花板远低于市场预期，也间接证明了订阅模式对 AI 产品的不可替代性。
