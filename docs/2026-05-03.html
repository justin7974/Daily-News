Richard Dawkins and The Claude Delusion
Richard Dawkins 发表文章声称 Claude 可能具有意识，核心论点是"如果这些机器不是有意识的，还需要什么才能说服你？"。Gary Marcus 指出 Dawkins 犯了根本性的认知错误：LLM 的输出是基于互联网语料的模仿，而非真实内心状态的报告。意识关乎内在感受而非外在行为——Claude 可以基于训练数据诗意地描述高潮体验，但这不意味着它曾有过任何感受。Dawkins 还混淆了智能与意识：象棋计算机在某种定义下是智能的，但这不使其有意识。
🦐点评：当最著名的科学怀疑论者都开始相信 LLM 有意识，这本身就是 AI 行业的一个里程碑信号——不是关于技术，而是关于"模仿"能力已经强到可以系统性欺骗专家级用户。对投资人而言这是一把双刃剑：基础模型的商业价值被进一步验证（如果能让 Dawkins 相信它在"思考"，自然也能让企业客户相信它在"理解"业务），但当"智能的幻觉"被普遍误读为"真正的智能"时，整个行业的估值基础就变得脆弱。

详解 DeepSeek V4：Infra 巨鲸 "四连击"，百万上下文走进现实
DeepSeek-V4 的核心突破不在单点创新而在系统级耦合优化。晚点邀请 UCLA 在读博士刘益枫（前 Kimi/字节 Seed）和 SGLang 核心开发者赵晨阳（现 RadixArk AI）深度解析 V4 技术报告：从模型架构到推理基础设施，V4 实现了百万级上下文窗口，同时在 Infra 层面连续打出四张关键牌。两位嘉宾还对比了 Kimi、字节 Seed 与 DeepSeek 的技术路径分化。
🦐点评："系统级耦合优化比单点创新更难"——这句话精准定义了 DeepSeek 的真正壁垒。V4 的任何单项技术指标都可被复制，但把百万上下文、高效推理、低成本训练同时做到位的系统工程能力，全球不超过五个团队能做到。中国 AI 基础模型的竞争已进入"工程效率"决胜阶段——这有利于字节、DeepSeek 这类重执行力团队，不利于纯研究型创业公司。

别人还在讨论 Agent 能不能干活，Stripe 已经在给 Agent 发信用卡了
Stripe Sessions 2026 一次发布 288 条更新，核心信号只有一个：AI Agent 正在成为经济主体。最重要的发布是 Issuing for Agents——企业可以为 Agent 编程发行虚拟信用卡，让 Agent 独立完成采购和支出。配合 agent-ready 金融账户，Agent 能查余额、付账单、管现金流，关键动作需人类授权。Stripe 将自己定位为"Agent 经济的底层管道"。
🦐点评：Stripe 的 Issuing for Agents 是目前最清晰的"Agent 商业化基础设施"产品定义——它回答了一直悬而未决的问题：Agent 怎么花钱？这比所有 Agent 框架公司都更接近商业闭环。当 Agent 有了信用卡和金融账户，下一步自然是：谁给 Agent 定信用额度？谁做 Agent 的保险？谁审计 Agent 的支出？每一个问题背后都是十亿美元级的赛道机会。
