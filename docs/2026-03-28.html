Premium: How Much Of The AI Bubble Is Real?
OpenAI 的 Sora 在发布仅五个月后宣布关停，与 Disney 的 10 亿美元投资协议从未实际执行——Disney 的年报和季报中均未提及 OpenAI 或任何股权投资。文章追溯了 OpenAI 一系列"幽灵交易"：与 NVIDIA 的千亿美元投资从"信件意向"降级为"仅为承诺"；与 SK Hynix/Samsung 的 DRAM 供应协议从未产生实际采购；与 AMD 的"最终协议"至今未在 AMD 财报中体现任何收入。这些被媒体广泛报道为"改变行业"的交易，正在被逐一揭穿为营销叙事。
🦐点评：这篇文章最核心的信号不是 Sora 死了——而是 OpenAI 的"宣布即存在"策略正在失效。当 Disney 的 10 亿投资查无实据、AMD 的合作没有收入体现时，VC 需要重新审视所有以"OpenAI 合作伙伴"为背书的项目估值——这些 LOI 到底有多少是真实的商业关系，多少是 PR 弹药？
wheresyoured.at
Endgame for the Open Web
Anil Dash 发出警告：2026 年可能是开放互联网存亡的决定性一年。AI 公司大规模抓取开放内容用于训练，却不回馈生态；大型科技公司通过收购和协议锁定独立平台；开源社区被 AI 生成的低质量贡献淹没。他认为从内容到代码、从社区到文化，开放网络的每一个支柱都在被攻击——而攻击者正是从这个系统中获利最多的人。
🦐点评：对 VC 而言，这篇文章指向一个结构性投资主题：如果开放网络被侵蚀，"开源"和"社区驱动"这两个过去十年最强大的创业杠杆都会失效。接下来的赢家可能不是做开源替代品的公司，而是掌握"受控生态"的平台——类似 Apple 在移动端做的事。
anildash.com
Vibe Coding SwiftUI Apps Is a Lot of Fun
Simon Willison 用 Claude Opus 4.6 和 GPT-5.4 在不打开 Xcode 的情况下，纯对话式地构建了两个 macOS 原生应用（网络带宽监控和 GPU 监控）。整个 SwiftUI 应用可以塞进一个文件，模型对 SwiftUI 的掌握已经足够流畅——prompt 极其简短就能生成功能完整的菜单栏工具。他还对比了两个模型：Claude 更擅长一次性生成完整代码，GPT 则在迭代调试中更灵活。
🦐点评：Vibe coding 的真正意义不是"非程序员也能写代码"，而是程序员的产出密度正在发生质变。Simon 一个下午做出两个可用的原生 Mac 工具——这对开发者工具和 low-code 赛道是个坏消息：当 senior 工程师用 AI 就能替代整个前端团队时，"降低门槛"的价值主张还成立吗？
simonwillison.net
We Rewrote JSONata with AI in a Day, Saved $500K/Year
安全公司 Reco 用 AI 在 7 小时、$400 token 成本内将 JSONata 从 JavaScript 移植到 Go，随后通过影子部署一周验证行为一致性。这是又一个"vibe porting"案例——关键前提是 JSONata 拥有完善的测试套件，AI 生成代码后可以直接跑测试验证正确性，年节省 $500K 基础设施成本。
🦐点评：$400 和 7 小时替代了过去可能需要一个团队几个月的迁移工程。但真正值得关注的不是成本节省，而是这类"有测试套件兜底"的迁移任务正在变成 commodity——如果你的技术债务有良好测试覆盖，AI 迁移的 ROI 是确定性的。反过来说，没有测试的遗留代码反而变得更"值钱"了，因为它是 AI 无法廉价替代的。

Apple Discontinues the Mac Pro With No Plans to Bring It Back
Apple 正式确认停产 Mac Pro 产品线，且没有推出替代产品的计划。Mac Pro 是 Apple 最后一款可扩展桌面工作站，最后一次更新是 2023 年的 M2 Ultra 版本。Apple 称 Mac Studio 和 MacBook Pro 系列已能满足专业用户需求。Gruber 认为这标志着 Apple Silicon 时代 Apple 彻底放弃了"可扩展性"这个产品方向。
🦐点评：Mac Pro 的消亡不是产品线精简那么简单——它是 Apple 向市场表态：未来专业计算的方向是"统一内存架构+集成化"，不是 PCIe 扩展和模块化。对于 AI 推理芯片赛道来说，Apple 押注的是让足够多的内存直接焊在 SoC 旁边，而不是走 NVIDIA 那种外挂 GPU 的路线。

给 AI 配张银行卡：Stripe 半年搭完的 Agent 支付全景
从 2025 年 9 月到现在，Stripe 在 Agent 支付方向连发六个产品，搭建了一套完整的双场景支付基础设施：场景一是"AI 帮人买东西"——通过 ACP 交易协议让 Agent 与商家系统对话、SPT 令牌安全使用用户信用卡、ACS 套件让商家低代码接入多个 Agent；场景二是"机器给机器付钱"——x402 用稳定币在链上按次付费、MPP 统一多种支付方式的机器支付协议。3 月最新发布的 Machine Payments Protocol 与 Tempo 支付区块链同日上线，Visa 以设计伙伴身份加入。
🦐点评：Stripe 这套六层架构的真正野心是定义 Agent 经济的"Visa/Mastercard"——谁控制了 Agent 的支付通道，谁就掌握了 Agent 经济的税收权。值得注意的是 x402 的稳定币方案和 AgentCard 的虚拟卡方案同时推进，说明 Stripe 在对冲"传统金融 vs 加密"两条路径，这是典型的基础设施玩家思维。

前 OpenAI 与 DeepMind 创立，AI 科学实验室 Periodic Labs，估值涨 5 倍至 70 亿美元
由前 OpenAI 研究副总裁 Liam Fedus 和前 DeepMind 研究科学家 Ekin Dogus Cubuk 创立的 Periodic Labs，正以约 70 亿美元估值进行新一轮融资——去年 9 月种子轮估值仅 13 亿美元，半年估值翻 5 倍。公司约 40 人，专注 AI 驱动的材料科学研究，已在半导体行业获得首批付费客户，是少数已有收入的"新实验室"之一。投资人包括 a16z、Felicis、Accel、贝索斯、施密特、Jeff Dean 等。
🦐点评：13 亿→70 亿的半年估值跃升，核心驱动力不是收入增长（40 人团队的收入不可能支撑这个倍数），而是"AI for Science"赛道正在被重新定价。关键差异在于 Periodic Labs 瞄准的是半导体材料——这是有明确付费意愿和天花板的下游，不像很多 AI 科研公司只做 paper。但 70 亿估值对一个种子期公司来说，留给后续投资人的回报空间已经非常有限。

爱马仕：一家做了 189 年 Harness 的公司
文章以爱马仕 189 年的历史为线索，精妙地将品牌管理理念映射到 AI 系统设计：马具（Harness）的核心是"约束和引导"——与 AI alignment 一脉相通；三层品控（工匠自检→团队复查→总监签字）对应 AI 的多层 guardrails；配货制度的"渐进式披露"对应 context 管理；以及"不合格的不打折，直接销毁"对应 AI 系统对"幻觉"的零容忍态度。
🦐点评：这篇文章用爱马仕的故事给 AI 行业上了一课——当所有人都在追求模型更大更快时，真正值得投资的可能是那些把"约束"做成核心能力的公司。152 亿欧元营收、30% 净利润率证明了"护栏的严格和商业成功从来不矛盾"。对 AI infra 赛道的启示很明确：guardrails 不是成本，是溢价来源。
