The Energy Situation, Explained for Tech People
波斯湾是全球能源贸易的 us-east-1，霍尔木兹海峡是最关键的单点依赖——一旦长期关闭，冲击将从油价蔓延至食品、化肥、航运、制造业乃至政治稳定。文章用技术人熟悉的框架解释能源危机：中东石油的生产成本全球最低（沙特 Ghawar 油田单井可采 1 亿桶，美国页岩井仅 50 万桶），但美国凭借资本市场的风险偏好硬是把 Permian Basin 从零做到超越沙特。1970 年代沙特开始布局石化下游，如今全球 70% 的石化产能集中在中东和中国，美国本土产能严重空心化。海峡关闭不只推高油价，更会引发化工原料断供的连锁反应。
🦐点评：这篇对 AI 投资人的核心启示不是能源本身，而是 AI 数据中心的选址和供电风险。美国正在疯狂建 GW 级数据中心，但电力供应链的脆弱性被严重低估——一次地缘冲击就可能让算力扩张计划集体延期。投硬件和基础设施的 VC 该认真算一下能源成本的 tail risk 了。

Premium: How Much Of The AI Bubble Is Real?
OpenAI 本周正式关闭 Sora，连同开发者版本和 ChatGPT 内的视频功能一并下线。更关键的细节是：2025 年 12 月宣布的迪士尼 10 亿美元投资从未真正发生——迪士尼 FY2025 年报和 Q1 FY2026 季报均未提及 OpenAI 或任何股权投资，Disney+ 上也找不到任何 Sora 视频。Sora 巅峰时每天烧掉高达 1500 万美元，生成的视频不过几秒钟且质量堪忧。作者痛斥主流媒体不加验证地传播 AI 营销叙事，给整个影视行业制造了不必要的恐慌。
🦐点评：Sora 之死是 AI 泡沫最具标本意义的案例——从"颠覆好莱坞"到五个月后彻底关停，中间没有任何商业化进展。迪士尼那笔"10 亿投资"连年报都没出现，说明大公司的 AI 合作公告正在变成纯粹的 PR 工具。VC 在看 AI 应用层 deal 时，需要区分"已经有用户在付费"和"只有新闻稿在发布"。

Endgame for the Open Web
Anil Dash 发出严厉警告：2026 年可能是决定开放互联网存亡的关键年份。AI 爬虫正在疯狂抓取发布者内容——每向网站发送 50 万次请求才导流 1 个用户，发布者承担了成本却得不到回报。LLM 平台用这些内容训练模型后几乎不提供原始链接。从内容到代码、从社区到文化，开放网络的每一层都在被科技巨头系统性攻击。他呼吁开源生态不能继续"business as usual"，必须像面对生存威胁一样反击。
🦐点评：对 VC 来说，开放网络的瓦解意味着流量获取成本将结构性上升。如果 AI 平台成为新的信息入口而不再导流，依赖 SEO 和内容营销的 SaaS 公司增长模型会被根本性颠覆。反过来，能帮发布者直接变现（付费订阅、社区）的工具赛道可能迎来新一轮机会。
anildash.com
We Rewrote JSONata with AI in a Day, Saved $500K/Year
Reco.ai 团队用 AI 在一天内将 JSONata（类似 jq 的 JSON 表达式语言）从 JavaScript 重写为 Go 实现，年省 50 万美元。关键在于 JSONata 已有完善的测试套件，AI 用 7 小时、400 美元 token 成本完成初版，随后通过 shadow deployment 一周并行验证新旧实现的一致性。Simon Willison 将这类操作归类为"vibe porting"——用 AI 将成熟项目在语言间移植。
🦐点评：这才是 AI 编程当前最实际的 ROI 场景——不是从零写新产品，而是把有完整测试覆盖的旧代码移植到更高效的语言。400 美元 + 1 天换 50 万美元/年，这个 ROI 让任何有大量 legacy code 的公司都该立刻评估哪些模块可以 vibe port。测试套件的完备度成了 AI 编程价值的瓶颈，没测试的代码连 AI 也救不了。
simonwillison.net
Vibe coding SwiftUI apps is a lot of fun
Simon Willison 在 128GB M5 MacBook Pro 上用本地 LLM 进行 vibe coding，替代 Activity Monitor 写了一套 SwiftUI 性能监控工具。新笔记本跑本地大模型的能力让他直接绕过 API 依赖，纯本地完成从需求到成品的全流程。文章展示了 M5 芯片在本地推理场景下的实用性已达到可替代云端 API 的水平。
🦐点评：M5 + 128GB 统一内存跑本地 LLM 做 vibe coding，这个组合标志着"开发者 AI"从 SaaS 订阅模式开始向本地化迁移。如果顶配笔记本就够用，Cursor/Windsurf 们的 pricing power 会被压缩。Apple Silicon 的 AI 护城河不在消费者端（Siri 依然拉跨），而在开发者生产力工具这个高客单价场景。

给 AI 配张银行卡：Stripe 半年搭完的 Agent 支付全景
从 2025 年 9 月到 2026 年 3 月，Stripe 密集推出六层 Agent 支付产品。两大场景：一是 AI 帮人买东西（ACP 交易协议 + SPT 支付令牌 + ACS 商家接入套件），ChatGPT Instant Checkout 已接入 Etsy 和百万 Shopify 商家；二是机器给机器付钱（x402 链上稳定币按次付费 + MPP 统一机器支付协议），Tempo 支付区块链同步上线。SPT 最新扩展支持 Mastercard Agent Pay、Visa Intelligent Commerce 以及 Affirm/Klarna 先买后付。Stripe 自称是唯一同时支持 Agent 网络令牌和 BNPL 令牌的支付提供商。
🦐点评：Stripe 这套六层架构的真正野心不是"帮你买鞋"——那只是演示。x402 + MPP 构建的"机器给机器付钱"才是杀手级场景：当 Agent 能自主调用付费 API、自主采购计算资源，支付基础设施就变成了 Agent 经济的"血管系统"。现在看谁能做中国版的 Agent 支付协议，这可能是下一个 Stripe 级别的机会。

爱马仕：一家做了 189 年 Harness 的公司
用爱马仕 189 年的品牌史做了一次精妙的类比：马具（Harness）的核心功能是"约束和引导，让力量朝正确的方向释放"——这恰好是 AI Alignment 的本质。鞍针缝法的双线容错是 fault tolerance，三层品控（工匠自检-质量复查-总监签字）是 guardrails，配货制度是 progressive disclosure，每件产品的可追溯性是 trace data。最反直觉的洞察：爱马仕每代继承人都在"砍掉上一代觉得很聪明的产品线"——build to delete。
🦐点评：这篇表面写奢侈品，实际是写给 AI 从业者的方法论。"材料越好对工艺要求越高"这句话翻译过来就是：模型能力越强，对齐和护栏的设计标准越高。当下 AI 公司普遍在做"加法"——加功能、加模态、加场景，而爱马仕的故事提醒我们，最持久的竞争力可能来自"减法"和克制。
