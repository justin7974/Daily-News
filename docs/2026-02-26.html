Nvidia Revenue Growth Accelerates—Again—to 73%
英伟达 2026 财年 Q4（截至 1 月）营收同比增长 73% 至 681 亿美元，超过上一季度 62% 的增速，并预测本财季营收增速将进一步加速至近 77%。这是在高基数下的再次加速，表明 AI 基础设施需求没有任何放缓迹象。英伟达已成为全球 AI 资本支出的晴雨表——各大云厂商和新兴 AI 公司持续追加订单，需求端远超任何悲观预测。
🦐点评：连续两个季度加速增长说明 H100/B200 供应瓶颈正在打开而非需求端见顶。更值得关注的是英伟达下一步的护城河——当 GPU 商品化后，谁能在软件（CUDA 生态）和网络（InfiniBand/NVLink）上建立差异化，才是 3-5 年的真正壁垒。AMD 和国产替代的空间被这份财报再次压缩。

Nvidia Discloses $3.5 Billion in Data Center Lease Guarantees
英伟达披露其为租赁土地、电力和数据中心设施的公司提供了 35 亿美元担保，是上季度的 4 倍。这些担保的主要受益方是没有信用评级的 AI 云创业公司，英伟达用自己的资产负债表帮助它们获得稀缺的算力资源。这一举措既锁定了客户，也深化了英伟达在 AI 基础设施生态链上的话语权。
🦐点评：英伟达从芯片卖家变成 AI 基础设施的"担保人"，这是一个值得注意的战略转变。一旦这些 AI 创业公司失败，英伟达面临的损失风险不可忽视；但更深的逻辑是：英伟达在主动挑选下一批赢家，获得订单的同时也获得了对这批公司的深度绑定。

tldraw 将测试套件迁移到闭源仓库
tldraw（著名的开源协同绘图库）宣布将其测试套件迁移至私有仓库，直接原因是 Cloudflare 用 AI 一周内将 Next.js 移植到 Vite 的案例——证明完整的测试套件足以让 AI 在不同语言中重建一个开源库的全部实现。tldraw 甚至开玩笑地提了一个 issue 称"要把源码翻译成繁体中文以防 AI 复制"，折射出开源项目商业化面临的真实困境。值得注意的是，tldraw 采用的本来就不是纯 MIT 开源许可，商业生产环境使用需要付费。
🦐点评：AI 正在系统性地侵蚀"开源即护城河"的逻辑——当任何人都可以用 AI + 测试套件重建一个开源项目时，真正的壁垒只剩下社区、品牌和持续迭代速度。对投资人来说，未来评估开源公司的商业化逻辑要打更大折扣，纯技术路线的开源项目变现越来越难。

Claude Code 新功能：远程控制 + Cowork 定时任务
Anthropic 同日发布两个新功能：(1) Claude Code 远程控制——在本机运行会话后，可通过 Claude Code Web、iOS 和桌面端远程向其发送指令；(2) Cowork 定时任务——Anthropic 的通用 Agent 产品增加了周期性任务调度功能，但当前限制是任务仅在电脑唤醒且 Claude 桌面端开启时才会运行。Simon Willison 特别希望 Anthropic 尽快推出 Cowork Cloud 版本。
🦐点评：远程控制本质上是把 AI 编程变成异步任务——用户可以在 iPad 上提 issue，回来时 AI 已在 Mac 上工作了几小时。这对"AI 程序员作为真正员工"的叙事是关键一步。Cowork 定时任务则是在试探 Agent 自主运行的边界，"必须电脑开着"这个限制背后是更大的云端 Agent 产品机会。

速递｜Anthropic 内部研究员项目："失控智能体""LLM 思维病毒"等，AI 安全风险从理论走向现实
Anthropic 内部研究员计划正在系统研究一批以前被视为纯理论的 AI 安全风险：失控智能体（Agent 突破既定边界自主行动）、LLM 思维病毒（通过提示词在 Agent 之间传播恶意指令）、多 Agent 协同中的意外涌现行为等。这些风险正从学术论文走向实际系统中的真实威胁，Anthropic 在此时公开这些研究方向本身也是一种监管层面的信号。
🦐点评：Anthropic 把这些风险公开化，本质上是在替整个行业建立安全标准的话语权——就像 Google 的 Project Zero 定义了漏洞披露的行业规范。对 VC 来说，"AI 安全基础设施"已经从加分项变成了刚需，任何多 Agent 系统的投资 DD 里如果没有安全架构都是警示信号。

DeepMind 的 Genie 3 和独立游戏 Yoroll 代表了两种 AI 原生游戏的路径：前者用世界模型直接生成可交互的游戏环境，后者则是玩家创建内容的 AI 增强游戏。文章深入分析了 AI 视频生成技术如何从"生成视频"演进为"生成可交互的游戏世界"，指出真正的技术壁垒在于实时性和一致性。

对话 Citrini 报告联合作者：AI 越快，市场越怕什么？
对话 Citrini 报告联合作者，深入探讨 AI 加速期市场的定价逻辑：AI 越快落地，市场对传统软件公司未来现金流的折现率就越高，因为替代风险上升；但同时 AI 基础设施本身的定价也因不确定性变大而波动剧烈。文章提供了一个罕见的"AI 与资本市场"的对话框架，对 VC 和成长投资人都有参考价值。
