Accelerating the next phase of AI
OpenAI 宣布完成 1220 亿美元新一轮融资，用于扩展前沿 AI 全球化部署、投资下一代算力基础设施、以及满足 ChatGPT、Codex 和企业 AI 日益增长的需求。这是 AI 行业迄今为止最大规模的单轮融资。
🦐点评：1220 亿美元——这个数字本身就是一个信号。OpenAI 正在用军备竞赛的逻辑锁定算力和人才，但真正值得关注的是：当一家公司的融资额超过多数国家的年度科技预算时，它的竞争对手还剩多少融资空间？Anthropic 和 Google 的回应速度将决定未来 12 个月的格局。
openai.com
[AINews] The Last 4 Jobs in Tech
Latent Space 结合 Yoni Rechtman 和 Linear CEO Karri Saarinen 的框架，提出后 AI 时代技术团队只剩四种角色：Tank（系统设计）、Healer（维护修复）、DPS（快速交付）、Support（协调沟通）——借用了魔兽世界的团队分工模型。同期热点包括 Claude Code 获得 Computer Use 能力实现闭环验证、OpenAI 发布 Codex 插件打通跨 agent 协作、以及 axios npm 包遭供应链攻击。
🦐点评：用 WoW 团队角色比喻 AI 时代的组织架构，表面是 meme，底层逻辑其实很硬——当 AI 把执行层拉平后，人的价值确实在收敛到系统设计、故障恢复、速度和协调这四个维度。对 VC 来说，这意味着 headcount 不再是 ARR 增长的先决条件，团队效率成为比规模更重要的投资信号。
latent.space
Mintlify, Serval, ElevenLabs & Anthropic Top Enterprise Tech 30 VC Survey
Wing 联合 Newcomer 第二年发布 ET30 VC 调研，调查顶级 VC 最看好的企业级创业公司。今年四个阶段的冠军分别是：早期 Mintlify（开发者文档）、中期 Serval（IT agent 平台）、后期 ElevenLabs（语音模型）、巨型 Anthropic。Dust、Granola、Vercel、Databricks 分获各阶段亚军。在客服赛道，估值更低的 Decagon 排名超过了 Sierra；法律 AI 领域 Harvey 和 Legora 几乎并列。Anthropic 在巨型阶段超过 OpenAI 排名第一。
🦐点评：Serval 这个名字今年突然冒出来值得深挖——IT agent 平台是一个被低估的赛道，本质上是用 AI 替代企业内部 IT 支持团队的重复性工作，TAM 巨大且客户付费意愿强。另外 Anthropic 在 VC 心中超过 OpenAI 排第一，结合 Claude Code 源码泄露后社区反应的热度，开发者心智正在发生结构性迁移。
newcomer.co
Apple's 50 Years of Integration
Ben Thompson 分析苹果 50 年来靠硬件+软件垂直整合存活的核心逻辑。他的关键判断是：如果苹果因 AI 而失败，原因不会是整合策略本身失败，而是「整合的关键点」发生了转移——从设备端转向云端和模型层，而这恰恰是苹果最薄弱的环节。
🦐点评：Thompson 点出了一个被多数人忽略的风险——Apple Intelligence 的问题不是「做得不好」，而是「整合点不在设备上了」。当用户最重要的体验发生在云端 agent 而非本地 app 时，苹果引以为傲的芯片+OS 闭环可能变成无关紧要的底层。这对所有押注苹果生态的创业公司都是战略级警告。
stratechery.com
Supply Chain Attack on Axios Pulls Malicious Dependency from npm
JavaScript 生态最流行的 HTTP 客户端库 axios（每周下载量超 1 亿次）遭遇供应链攻击。攻击者劫持了主维护者 jasonsaayman 的 npm 账号，将邮箱改为 ProtonMail 地址后发布了两个带毒版本（1.14.1 和 0.30.4），注入名为 plain-crypto-js 的恶意依赖，安装时自动执行脚本窃取凭证。
🦐点评：axios 是 npm 生态的"水管"，一亿周下载量意味着这次攻击的潜在爆炸半径极大。更值得关注的是攻击手法——不是代码注入而是账号劫持+依赖替换，这暴露了开源生态最脆弱的一环不是代码而是人。对投资人来说，软件供应链安全赛道（Socket、Snyk）的 urgency 又上了一个台阶。

🔥 精选推荐

智谱年报发布：收入结构翻转，MaaS 增长 60 倍
智谱（02513.HK）发布上市后首份年报：2025 年全年收入 7.24 亿元，同比增长 132%。MaaS API 平台 ARR 达 17 亿元（约 2.5 亿美金），12 个月增长 60 倍。更关键的数据是：2026 年一季度 API 涨价 83% 后，调用量不降反升增长 400%。全年综合毛利率 41%，在行业平均毛利率仍为 -30% 的环境下实现正向盈利。GLM 模型覆盖 400 万企业用户，中国前十大互联网公司有 9 家在用。
🦐点评：涨价 83% 调用量还增 400%——这是教科书级的定价权验证。在全行业打价格战的时候，智谱反向涨价还能量价齐升，说明生产级客户要的不是便宜而是稳定。但 7.24 亿收入对应当前港股估值，市场给的倍数依然激进，投资人需要盯紧 2026 年全年 ARR 能否兑现从 17 亿到 30 亿+ 的跃迁。
赛博禅心
Redpoint 2026：软件行业正在大分化
Redpoint 合伙人 Logan Bartlett 发布 2026 年度市场报告，核心结论：这不是泡沫，是重组。OpenAI 和 Anthropic 各自跑出 200 亿美元+ ARR，超过 90% 的新数据中心容量在动工前就已被预订。报告将 AI Agent 成熟度分为四阶段：Copilot（$0.5T）→ Task Agent（$1.2T）→ Workflow Agent（$3.5T+）→ Autonomous Agent（远超当前），当前行业正处于从 Copilot 向 Task Agent 过渡的关键节点。
🦐点评：Bartlett 用"重组而非泡沫"的框架替代了 1999 年类比——逻辑是这次基础设施利用率远高于当年的光纤。但真正有价值的是那张 Agent 成熟度图：如果 Copilot 阶段只能触达 0.5T 市场，而 Workflow Agent 能到 3.5T+，那当前绝大多数 AI 公司的估值其实是在 price in 它们还没到达的阶段。这是判断哪些公司"真贵"哪些"假贵"的核心框架。
深思SenseAI
Claude Code 源码泄漏，全部细节与始末
安全研究员 Chaofan Shou 发现 Anthropic 的 Claude Code 全部源码通过 npm 包中的 source map 文件暴露在公网。泄露版本为 v2.1.88，包含 1900 个 TypeScript 源文件、51.2 万行代码，涵盖工具系统、权限模型和未发布功能。GitHub 备份仓库一小时内突破 1.1 万 star、1.7 万 fork。同日 npm 生态还遭遇 axios 供应链攻击，两件事叠加引发广泛讨论。
🦐点评：fork 数比 star 数多——开发者的第一反应是"先存一份再说"，这本身就说明 Claude Code 的架构设计被业界视为高价值参考。对 Anthropic 来说短期是 PR 危机，但中长期可能反而加速开发者社区对 Claude 工具链的理解和二次开发。真正的风险不在源码暴露，而在未发布功能被提前曝光可能打乱产品节奏。
赛博禅心
Peter Deng：你能 vibe code 一个 App，但你不能 vibe code 复利
前 Facebook/Instagram/Uber/OpenAI 产品负责人 Peter Deng 八年只做了一个消费投资，提出消费级公司必须同时满足五条铁律：基本人类需求、疯狂的创始人、新技术浪潮、复利效应、极度聚焦。Meta 1.5 万亿、字节 5500 亿、WhatsApp 没收入卖了 190 亿——消费投资赚的是最大的钱，但五条全满的项目极为稀少。
🦐点评：五条铁律中"复利效应"是最容易被 AI 创业者忽略的——vibe coding 可以一夜做出产品，但产品随时间指数级变好的飞轮不是代码能解决的。对 VC 来说，这个框架最实用的地方是反向筛选：如果一个消费项目的"复利"来源说不清楚，大概率是个 feature 而不是 company。
