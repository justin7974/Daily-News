White House Grants Access to Anthropic's Mythos Model to 100+ U.S. Institutions; Fable Still Shut Down
美国商务部周五撤销对 Anthropic Claude Mythos 5 的封禁，允许其向 100 多家美国机构（含大企业和政府机构）发布。两周前政府刚因亚马逊等公司警告该模型可能被"越狱"用于恶意用途，对 Mythos 实施出口管制、导致 Mythos 与 Fable 5 双双下架。商务部长 Lutnick 在信中称已确认"适当的保障措施到位"，可让"特定可信伙伴"访问，并称 Anthropic 承诺与政府共同制定模型发布的协议与标准；信里只字未提仍处关闭状态的、更弱的 Fable 5，知情者称正朝放开方向走但时间表不明。同一天，竞对 OpenAI 也仅向少数政府批准的伙伴发布了 GPT-5.6。
🦐点评：真正的拐点不是 Mythos 解封，而是 Lutnick 那句"trusted partners"和信里的 Annex A 白名单——这是美国第一次以行政信件的形式给前沿模型发"准入许可证"，前沿 AI 正从公开 API 商品变成需逐家审批的受控战略资产。对 VC 的二阶含义：模型层护城河不再只看能力，而看谁拿得到政府白名单和主权/关键基础设施客户关系，纯转售 API 的应用层最先被这套许可制掐住脖子。两周从封禁到解封也说明实验室在博弈里几乎没有议价权——谁能把"和商务部每日谈判"做成常态化能力，谁就拿到下一轮的卡位。

OpenAI GPT-5.6 Sol / Terra / Luna — restricted to trusted partners
GPT-5.6（Sol 旗舰 / Terra 均衡 / Luna 低成本）不是常规公开发布，而是"受限预览"——初期仅向 Codex 与 API 中约 20 家政府审核过的可信伙伴开放，OpenAI 明确说这是"应美国政府要求"。Sol 在部分编码 agent 任务上超过 Anthropic Mythos，但 OpenAI 刻意强调它在网络安全上不及 Mythos、未跨过其 Preparedness Framework 的 Cyber Critical 阈值（能找到漏洞与利用原语，但未能在测试条件下自主造出完整攻击链）。Altman 称原计划更大范围发布，因政府要求改为有限预览，正努力尽快转 GA。评论者普遍把这解读为前沿发布正变成"可信伙伴优先"的政府中介式部署。
🦐点评：这条和 Mythos 解禁是同一枚硬币的两面——OpenAI 主动把"我没 Mythos 那么会黑客"写进发布稿，说明在新监管框架下"能力不够强"反而成了拿到放行的卖点，这是前沿竞赛激励第一次被监管逆转。定价信号同样关键：Terra 性能对标 GPT-5.5 却便宜一半，每代价格塌缩没停；叠加只放给约 20 家的稀缺性，等于人为制造了"买得到 token"和"买不到"两个市场。押注 API 普惠分发的创业公司要重算账——你的供给侧现在握在华盛顿手里。

Salesforce Employees Worry Over Anthropic's Invasion of Slack
Anthropic 本周推出面向企业的高调产品 Claude Tag——一个常驻 Slack 群聊、能看到并记住群里全部讨论的"AI 队友"。讽刺的是，Slack 母公司 Salesforce 部分员工对此感到困惑：Salesforce 自己还在社交媒体上帮这款来自 Anthropic 的产品做推广。（基于 The Information 报道开头，正文付费墙）
🦐点评：看点是平台层与模型层的边界正被 Anthropic 直接打穿——Slack 这种坐拥企业沟通数据的系统级入口本是 Salesforce 的护城河，结果 Anthropic 用一个常驻群聊、记住一切的 agent 直接在别人地基上盖楼，Salesforce 员工还在帮它吆喝。对 VC 的判断：拥有分发渠道 ≠ 拥有 AI 价值，平台方若不能自建够强的模型，最肥的"company context"反而会被模型方收割；这也是今年所有 SaaS 入口公司都在拼命自研 agent 的原因——晚一步，自己的用户数据就成了别人的驻场地。
theinformation.com
Coinbase Cuts AI Spending in Half As Usage Rises, Armstrong Says
Coinbase 的 AI 开支"砍掉近一半"，同时 token 用量还在上升——CEO Brian Armstrong 周五晚在 X 上说，关键控本手段之一是默认改用中国公司的开源权重模型。（The Information，正文付费墙）
🦐点评：一家美国上市公司 CEO 公开承认"用中国开源模型把账单砍半"，比任何 benchmark 都更能说明开源模型的渗透已从 demo 走到生产财务决策。把它和今天政府收紧美国前沿模型放在一起看，张力就出来了：华盛顿越是把 Mythos/GPT-5.6 变成审批制稀缺品，企业越有动机倒向 DeepSeek/GLM 这类开源替代——管制反而在替中国模型做分发。对看 infra 的基金，这是"前沿能力溢价正被开源 + 成本压力快速抹平"的实锤，纯卖前沿 API 调用的毛利故事越来越难讲。

FT: Apple Is Lobbying to Buy Memory Chips From Blacklisted Chinese Company CXMT
据 FT（六名知情人士），苹果正游说特朗普政府，争取从中国内存厂商 CXMT 采购内存芯片的许可——CXMT 因被指与解放军有关联而被五角大楼列入黑名单。苹果目前并未被禁止向 CXMT 或另一家中国存储厂 YMTC 采购，但五角大楼的黑名单制造了实质的政治阻力。（DF 转引 FT，正文付费墙）
🦐点评：苹果宁可去游说采购一家被五角大楼拉黑的中国内存厂，等于公开承认这轮 AI 数据中心引发的内存超级周期已经紧到让全球供应链一哥都缺货——这正是前一天苹果全线涨价的供给侧根因。地缘维度更微妙：当 AI 算力把美光/海力士产能全吸走、连苹果都被迫回头找 CXMT/YMTC，说明"去中国化"硬件供应链在内存这个卡脖子环节出现了反向松动的压力。对硬件与半导体赛道，内存涨价的明牌赢家是周期股，但真正的 alpha 在于谁能在中美都不待见的夹缝里拿到替代供给——CXMT 这类公司的战略价值正被动抬升。

🔥 精选推荐

都在劝你把活全甩给 AI，可做 jQuery UI 的那个人说：那是个伪命题
jQuery UI 与 Chrome DevTools 移动预览功能的作者 Paul Bakaus 发布新公司 Renaissance Geek 和产品 Impeccable（a16z 领投、与 GitHub 合作），并直接把"loopmaxxing"（把活全甩给 AI、让循环自己跑完）判为伪命题。他的核心论点是：AI 把"做出初稿"的地板抬高了，但卓越的天花板没变低；品味能被放大却无法被"装进瓶子"量产，因为 slop（设计垃圾）是会移动的靶子——2022 年的 slop 是紫色渐变发光按钮，2026 年变成暖米色背景配 Instrument Serif 斜体大标题，今天的解药就是明天的 slop。他还指出设计与工程之间的"交接墙"正在塌，而主流的 chat 交互是被动的，远不如给你画布/时间线的创意软件。
🦐点评：把 Bakaus 这套话翻成投资语言——当 AI 把"能用的初稿"变成零边际成本商品，价值就从"产能"整体迁移到"判断与品味"这层稀缺资本，这正面狙击了今年一票"全自动 agent 工厂"叙事的估值假设。他自己做的反而是"除 slop"工具 Impeccable + a16z 背书，赌的是"放大人的意图"而非"替代人"，给 AI 创意/设计工具指了一条和 Cursor 式全自动不同的路：不卖自动化，卖让人做选择的界面（画布而非聊天框）。对看 AI 应用层的基金，"slop 是移动靶"是个狠提醒——任何靠模型默认审美吃饭的产品，护城河会随大众审美迁移而蒸发。
